# Swerve

## Introduction

This repository contains the downloader of all roadrunner files used by the Swerve project.

## Usage

For Linux and macOS users, run the following command in your terminal:

```bash
git clone https://github.com/BluePowerRobotics/Swerve.git
cd Swerve
chmod +x ./download
./download
```

For Windows users, run the following commands in your Command Prompt:

```cmd
git clone https://github.com/BluePowerRobotics/Swerve.git
cd Swerve
./download.bat
```

or use Git Bash:**(better)**

```bash
git clone https://github.com/BluePowerRobotics/Swerve.git
cd Swerve
chmod +x ./download
./download
```

After executing the appropriate commands for your operating system, all necessary dependencies will be downloaded to the Swerve project directory.

Use *Android Studio* to open the **SwerveDrive** directory and follow the direction in README.md.
